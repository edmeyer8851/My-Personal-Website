Working Snake game I made with a Youtube code-along:

After hours upon hours trying to code the game myself, I soon realized that some of my earlier code was becoming ridiculously complex to adjust in order to get a finished, working product.

I wanted to display some working game on this page and more importantly, I was now very curious as to what other solutions were out there for the same problems I had been trying to solve. I coded things a certain (less than ideal) way during my attempt, but how would someone who actually knows what they're doing write this program?

I turned to my good friend Youtube, and found a How to code the Snake game in Javascript code-along video. I learned about a bunch of new concepts while following along with this video: CSS grid, requestAnimationFrame() (this is how the game progresses on it's own), and using Javascript modules to break code down into separate files.
